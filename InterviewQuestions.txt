4)What is repositary pattern?
Ans.
Repositary pattern helps to abstract and centralize the data access logic thus providing a clean and consistent interface for CRUD operation on models.

5) What is DAL(Data Access Layer)?
Ans.
DAL focuses on the technical details of data access while repositary provides higher level of abstraction over DAL.
DAL provides technical centric approach while repositaries provide Domain centric approach.

DAL -- Technical Details
Repositary -- High level abstraction over DAL


6) Explain generic repositary pattern?
Ans. 
It is an extension to repositary pattern.

Rather than creating separate interfaces/classes for each entity, you can just create a generic interface/class for all entities.
